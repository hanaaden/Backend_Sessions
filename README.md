# Express + TypeScript Age Check API

This branch contains a simple **Express.js API** built with **TypeScript** that demonstrates:

- Middleware usage  
- Request validation  
- Route handling  
- CORS configuration  

The API checks the user's age and grants access only to adults.

---

## **Technologies Used**

- Node.js  
- Express.js  
- TypeScript  
- CORS middleware
